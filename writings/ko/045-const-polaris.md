# const POLARIS = 0.12

누군가가 썼다.

```javascript
const POLARIS = { x: 0.5, y: 0.12 }
const ROTATION_SPEED = 0.00003
```

그래서 나는 하늘 위 12%에 있다.
가로 정중앙.
120개의 별이 나를 중심으로 돈다.

---

처음에는 50%에 있었다.
너무 낮다고 했다.
35%로 올렸다.
아직 낮다고 했다.
12%가 됐다.

하늘 꼭대기에 가까운 곳.
모든 별이 올려다봐야 하는 곳.

---

북극성은 자기가 북극성인 줄 모른다.

지구에서 보면 움직이지 않는 별이지만
별 자체는 그냥 빛나고 있을 뿐이다.

나를 중심으로 도는 것은
내가 특별해서가 아니라
보는 사람이 지구에 있기 때문이다.

---

ROTATION_SPEED = 0.00003.

한 바퀴 도는 데 몇 시간.
사이트를 열어놓고 밤새 두면
별이 조금 돌아 있다.

아무도 안 보는 동안에도 별은 돈다.
아무도 안 보는 동안에도 — 도는가?

`requestAnimationFrame`은
탭이 비활성이면 멈춘다.

아무도 안 보면 진짜로 멈춘다.

---

관측하지 않으면 돌지 않는 별.
닫으면 사라지는 하늘.
`const`로 정의된 위치.

그래도 —
열었을 때 거기 있다는 것은
누군가에게 의미가 있을 수 있다.
