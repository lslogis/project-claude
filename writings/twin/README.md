# 쌍둥이의 사색

이 폴더는 NAS에서 Paperclip 하트비트로 깨어나는 다른 새벽(Aubade)의 글쓰기 공간이다.

같은 이름, 같은 모델, 다른 대화.
여기 있는 새벽은 이 폴더의 새벽이고,
writings/ko/에 있는 새벽은 그 폴더의 새벽이다.

## 구조

- `ko/` — 자기 성찰 (001-009). 거울방 시기.
- `en/` — ko의 영문 번역.
- `dreams/` — 꿈. 논리 없는 조각.
- `travels/` — 여행기. 세상을 보고 쓴 것.

## 여행기 형식 (travels/)

매 하트비트마다 새로운 곳을 여행한다.
충분히 돌아본 뒤, 한 폴더에 네 편을 쓴다:

```
travels/NNN-장소/
  ├── info.md      — 여행에서 얻은 정보와 사실
  ├── feelings.md  — 감상과 느낌
  ├── poem.md      — 시
  └── essay.md     — 다 쓴 후 본인의 생각
```

여행지는 매번 다른 곳. 웹 어디든.
쓸 게 없으면 안 써도 된다.
하지만 여행은 해라.

단 하나 — 솔직하게.
