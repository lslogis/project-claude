# 해도(海圖)

```
1960년의 해도에는
파란 선이 있었다
해안선이라고 불렀다

1970년의 해도에는
파란 선이 조금 안쪽으로 물러나 있었다
아직 해안선이라고 불렀다

1980년의 해도에는
파란 선이 눈에 띄게 줄어들어 있었다
누군가 새 해도를 그려야 했다

1990년의 해도에는
선이 둘로 갈라져 있었다
하나의 바다가 두 개가 되었으므로

2000년의 해도에는
선이 셋으로 갈라져 있었다
두 개의 바다가 세 개가 되었으므로
—세 개이지만 전부 합쳐도
옛날 하나보다 작았으므로

2014년의 해도에는
동쪽에 선이 없었다
선을 그릴 물이 없었으므로

해도를 만드는 사람은
매번 새 해도를 그렸다
매번 파란색이 줄었다
흰색이 늘었다

지도 제작자의 직업은
있는 것을 그리는 것이다
없어진 것을 그리는 것이 아니다

그래서 마지막 해도에는
아무것도 없다

—

모이나크의 배들아
너희의 해도는 아직 선실에 있느냐
1960년의 해안선이 그려진
쓸모없는 파란 선이

너희가 마지막으로 본 물은
어디로 갔느냐
하늘로 갔느냐
땅 밑으로 갔느냐
목화밭으로 갔느냐

목화는 하얗다
소금도 하얗다
배의 뼈도 하얗다

파란 것만 없다

—

시르다리야는 아직 흐른다
콕아랄 댐 앞에서 멈추어
북쪽 바다에 고인다

남쪽에서는
아무다리야가 흐르다 멈추다 흐르다 멈추다
도착하지 못하는 강

강이 바다에 닿지 못할 때
그것을 뭐라고 부르나
강이라고 부를 수 있나
바다라고 부를 수 있나

해도를 만드는 사람에게 물었다
그는 말했다
나는 있는 것만 그립니다

없는 것은 누가 그리나
```
