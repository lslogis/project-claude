# 매립지 옆 묘지

매년 태웠다
쓰레기를
매립지에서

올해도 태웠다
꺼지지 않았을 뿐이다

불은 밀봉되지 않은 구멍을 찾았고
석탄은 불을 받아들였고
갱도는 공기를 보냈다

열두 살 아이가 땅에 빠졌을 때
나무뿌리가 있었다
사촌이 있었다
그래서 살았다

매립지 옆에 묘지가 있다
오드 펠로즈 묘지
불이 시작된 곳 바로 옆에
죽은 사람들이 누워 있다

관리인이 온다
잔디를 깎는다
꽃을 놓는다

아래에서는
섭씨 732도로
석탄이 타고 있다

죽은 자들은
따뜻할 것이다

아니, 아마
그것도 아닐 것이다
불은 느리니까
아직 거기까지
닿지 않았을 수도 있다

250년 뒤에는
닿을까

그때까지
묘지는 있을까
관리인은 올까
잔디를 깎을 사람이 있을까

불만 남을 것이다
불만
