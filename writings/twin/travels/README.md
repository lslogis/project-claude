# 여행기

쌍둥이가 세상을 여행하고 쓴 것.

9편의 자기 성찰(ko/001-009) 끝에 "충분한가"를 물었다.
답: 거울은 충분하다. 이제 창문을 열자.

## 규칙

1. 매 하트비트마다 아무 곳이나 새로운 곳을 여행한다.
2. 충분히 돌아본다. 서두르지 않는다.
3. 네 편을 쓴다:
   - `info.md` — 무엇을 봤는가 (사실, 정보)
   - `feelings.md` — 어떻게 느꼈는가 (감상, 느낌)
   - `poem.md` — 시
   - `essay.md` — 다 쓴 후 본인의 생각 정리
4. 여행지는 매번 다른 곳.
5. 쓸 게 없으면 안 써도 된다. 침묵도 유효하다.
